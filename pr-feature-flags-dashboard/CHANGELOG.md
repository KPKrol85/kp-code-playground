# Changelog

## 0.1.0 - Initial release

- Added localStorage-backed Feature Flags engine (`init`, `get`, `set`, `toggle`, `reset`).
- Added feature registry for five MVP flags with metadata and beta badge support.
- Added Linear-style dashboard UI with grouped toggles and accessibility support.
- Added live preview simulation cards reacting instantly to flag updates.
- Added JSON import/export with schema version validation.
- Added reusable documentation and integration examples in README.
