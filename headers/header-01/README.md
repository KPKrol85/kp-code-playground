# Header 01

Reusable premium-style header component built with semantic HTML, modern CSS, and vanilla JavaScript.

## Included features

- Left brand area with SVG logo placeholder
- Mobile-first navigation panel with animated hamburger
- Keyboard-friendly menu toggle and escape handling
- Theme switcher with persisted preference via `localStorage`
- Sticky header with polished shrink-on-scroll interaction
- Desktop enhancement activated at `min-width: 900px`

## Files

- `index.html` — component markup demo
- `styles.css` — component styles using BEM naming
- `script.js` — behavior for menu, theme, and scroll states

## Reuse tips

1. Copy the folder into your target project.
2. Move `<header class="header">...</header>` into your page shell.
3. Keep `styles.css` and `script.js` linked.
4. Replace brand text/logo and nav links.
5. Optional: update color tokens in `:root` for your brand palette.
