# Axiom Construction

Strona typu landing page dla firmy budowlano-remontowej prezentująca ofertę usług, realizacje, opinie oraz dane kontaktowe. Projekt został przygotowany jako statyczna witryna zoptymalizowana pod SEO i doświadczenie użytkownika.

## Technologie
- HTML5
- CSS3
- JavaScript (Vanilla)

## Najważniejsze cechy
- Rozbudowane sekcje ofertowe i informacyjne
- Motyw jasny/ciemny z przełącznikiem
- Responsywny układ i zoptymalizowane assety
- Metadane SEO oraz schema.org (JSON-LD)

## Instrukcja uruchomienia
1. Otwórz plik `index.html` w przeglądarce.
2. (Opcjonalnie) uruchom lokalny serwer statyczny, aby przetestować Service Worker.
