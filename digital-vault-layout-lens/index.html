        Manual AI handoff

        AI-assisted review summary

        Layout Lens never contacts a model automatically and does not upload data. Prepare a session-only evidence package, copy it only if you choose where to send it, then paste the external AI model's JSON response for validation. Only the displayed deterministic evidence package is intended for the external model. Imported AI content remains AI-generated, session-only, and separate from scores, findings, recommendations, and reports.

        Prepare request

        The package can include manual score facts, reviewed counts, selected preset/rule pack/severity profile, category score facts, selected manual findings and notes, deterministic recommendations, and analyzer findings. It excludes raw HTML, raw CSS, preview content, project metadata, report metadata, storage internals, filters, browser details, and saved-project history.

        Senior frontend reviewer is the default preset.

        Privacy controls

        Conservative defaults include deterministic findings but exclude reviewer notes and snippets until explicitly enabled.

        No AI review request prepared.

        Session-only local activity: no requests prepared or copied. Layout Lens does not know provider price, token usage, network latency, or external model execution time.
