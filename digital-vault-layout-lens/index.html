    How to use this v1 checklist

    Review the UI manually, category by category. Set each rule to Not checked, Pass, Needs work, or Not applicable. The selected preset, active rule pack, severity profile, and rule statuses are saved locally in this browser only.

    How statuses affect the score

    • Pass: earns the rule’s weighted points.
    • Needs work: adds the rule to possible points, but earns 0 points.
    • Not applicable: counts as checked, but is removed from possible points.
    • Not checked: stays unreviewed and does not affect points yet.
