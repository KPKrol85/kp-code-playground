# Client Discovery Call Planner

A premium one-page planning tool from the **KP_Code Digital Vault** for running better client discovery calls before pricing a website or web project.

## Who this is for

- Freelancers offering website and web design services
- Small web studios handling discovery and proposal workflows
- Consultants who need a reusable intake framework for new client calls

## What it includes

- Structured call preparation checklist
- Client and project snapshot form fields
- Practical discovery question bank grouped by key business areas
- Interactive lead quality scoring tool (0-24 scale)
- Red flag reference list for risk identification
- Guided notes and blocker capture areas
- After-call action checklist with next-step guidance

## How to use it

1. Open `index.html` in any modern browser.
2. Complete the **Call Preparation Checklist** before the meeting.
3. Fill in the **Client & Project Snapshot** during the call.
4. Use the **Discovery Question Bank** to guide the conversation.
5. Score the lead in the **Lead Quality Scoring Tool**.
6. Record notes, risks, and required follow-up items.
7. Select your post-call actions and confirm the recommended next step.

## Suggested workflow

1. **Pre-call (10-20 mins):** review client context and prepare agenda.
2. **Live call (30-60 mins):** capture facts, priorities, and constraints.
3. **Post-call (15 mins):** score lead quality and map next action.
4. **Decision:** propose, request clarification, or decline based on fit.

## Customization notes

- Adjust scoring thresholds in `js/main.js` to match your qualification process.
- Edit question groups in `index.html` to fit your niche (e.g., SaaS, local business, ecommerce).
- Update visual tokens in `css/style.css` for your brand color system.
- Add additional action plan options if your sales process has more stages.
