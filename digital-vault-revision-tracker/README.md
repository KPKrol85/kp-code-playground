# KP_Code Digital Vault - Tracker Poprawek Klienta

Samodzielna, statyczna strona produktu do śledzenia próśb o poprawki od klienta, decyzji zakresowych, priorytetów, odpowiedzi i kosztów dodatkowych.

## Pliki

- `index.html` - Struktura strony produktu i treści.
- `styles.css` - Responsywne style mobile-first.
- `script.js` - Interakcje tabeli poprawek po stronie klienta i podsumowanie na żywo.
- `revision-tracker-template.md` - Szablon markdown do użycia offline/manualnie.

## Jak używać

1. Otwórz `index.html` w nowoczesnej przeglądarce.
2. Dodawaj prośby o poprawki przez formularz.
3. Przeglądaj wpisy w tabeli i panelu podsumowania.
4. Korzystaj z szablonu markdown w dokumentacji projektu lub aktualizacjach klienta.

## Uwagi

- Brak zewnętrznych bibliotek i frameworków.
- Dane są przechowywane tylko w bieżącej sesji.
- Dodatkowy koszt jest walidowany jako liczba nieujemna.
