# KP_Code Digital Vault - Revision Tracker

A standalone, static product page for tracking client revision requests, scope decisions, priority, responses, and additional costs.

## Files

- `index.html` - Product page structure and static content.
- `styles.css` - Responsive, mobile-first styling.
- `script.js` - Client-side revision table interactions and live summary calculations.
- `revision-tracker-template.md` - Reusable markdown template for offline/manual tracking.

## Usage

1. Open `index.html` in any modern browser.
2. Add revision requests with the form.
3. Review entries in the live table and summary panel.
4. Use the markdown template in project docs or client updates.

## Notes

- No external libraries or frameworks are used.
- Data is session-only and not persisted.
- Additional cost is validated as a non-negative number.
