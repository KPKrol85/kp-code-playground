Practical usage guide

  1. Log every client request immediately.
  2. Mark in scope only when it matches the agreed deliverables.
  3. Use response notes to document decisions and timelines.
  4. Confirm cost before implementing out-of-scope work.

Scope-control tips

  • Link each request to a contract section or approved brief.
  • Use "needs clarification" when requirements are unclear.
  • Escalate urgent requests with a confirmed turnaround window.
  • Review summary totals before weekly client check-ins.
