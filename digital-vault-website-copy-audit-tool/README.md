# Website Copy Audit Tool

A standalone KP_Code Digital Vault product for auditing website and landing page messaging quality. It helps you quickly evaluate clarity, trust, specificity, and conversion readiness using a practical worksheet plus an interactive scoring model.

## Who this is for
- Freelance web designers and developers
- Copywriters and conversion specialists
- Small web studios and agencies
- Business owners reviewing their own site copy

## What it includes
- One-page audit interface in static HTML/CSS/JS
- Structured sections for clarity, CTA, services, trust, and friction checks
- 5-second clarity test prompts
- Benefit-language rewrite prompts
- Quick-fix action planning fields
- Interactive 8-category scoring tool (0–24 total)
- Print-friendly layout for client-facing worksheets

## When to use it
- Before publishing a new service page or landing page
- During redesign or conversion optimization projects
- In discovery workshops with clients
- For regular copy quality checks during growth cycles

## How to use it
1. Fill out the **Audit Snapshot** for the page under review.
2. Complete the **5-Second Clarity Test** to check first-impression communication.
3. Audit hero messaging, CTA strength, offer descriptions, and trust proof.
4. Use the **Benefit Language** and **Vagueness/Friction** sections to rewrite weak copy.
5. Score each category from 0 to 3 in the **Interactive Scoring Tool**.
6. Use **Quick Fixes and Action Plan** to define immediate improvement actions.

## Scoring explanation
The tool includes 8 categories with a score of 0–3 each:
- Hero clarity
- Audience clarity
- Problem/solution fit
- CTA strength
- Service explanation
- Benefit language
- Trust signals
- Specificity

Maximum score: **24**

Score bands:
- **0–8:** Weak copy
- **9–15:** Decent but unclear
- **16–20:** Strong foundation
- **21–24:** Conversion-ready copy

## Suggested audit workflow
1. **First pass (5 minutes):** Fast scan + snapshot details.
2. **Second pass (10–15 minutes):** Complete checklists section by section.
3. **Third pass (5 minutes):** Score all 8 categories.
4. **Final pass (5 minutes):** Capture top issues and rewrites in quick fixes.
5. **Delivery:** Share as a worksheet or print/PDF with recommended next actions.

## Customization notes
- Update wording to match your niche (e.g., SaaS, local services, ecommerce).
- Add your studio branding colors in `css/style.css`.
- Extend scoring categories or recommendation rules in `js/main.js`.
- Duplicate and package as reusable client templates.
