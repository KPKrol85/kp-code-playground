# Strażnik Scope Creep

**Strażnik Scope Creep** to samodzielny mini-produkt KP_Code Digital Vault do szybkiej klasyfikacji próśb klienta i wyboru właściwej ścieżki odpowiedzi.

## Dla kogo

- Freelancerzy: web designerzy i developerzy
- Małe studia webowe
- Project leadzi prowadzący komunikację z klientem i kontrolę zakresu

## Co rozwiązuje

Prośby klienta często brzmią „niewinnie”, ale potrafią znacząco zwiększyć nakład, termin i budżet. To narzędzie daje spójny model punktowy, który pomaga podejmować spokojne i obronione decyzje zakresowe.

## Zawartość

- `index.html` — interfejs produktu i sekcje treści
- `styles.css` — styl KP_Code Digital Vault
- `script.js` — logika scoringu, kategorie wyników, reset i kopiowanie odpowiedzi
- `README.md` — przewodnik produktu

## Uwaga

Narzędzie wspiera decyzje i komunikację. Nie zastępuje zapisów umowy, konsultacji prawnej ani formalnych procesów change request.
