Intake Instructions for the Client

  • Provide original source files whenever possible (vector logos, editable docs, and export-ready media).
  • Avoid low-resolution screenshots as final project assets.
  • Group files clearly by category (brand, copy, media, access, legal).
  • Use clear naming conventions so each file can be matched to its page or function.
  • Share technical access through secure methods and verify credentials are current.
  • Confirm ownership and usage rights for all images, videos, fonts, and media before submission.
