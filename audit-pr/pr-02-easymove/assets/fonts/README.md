# Local font assets

This project defines local `@font-face` rules for the `Inter` family in `css/fonts.css`.

Place the following files in this directory:

- `Inter-400.woff2`
- `Inter-500.woff2`
- `Inter-600.woff2`
- `Inter-700.woff2`

All files are expected to be normal style (`font-style: normal`).
