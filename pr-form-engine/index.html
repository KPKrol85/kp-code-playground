Universal Form Engine

Reusable, accessible, and product-ready form foundation for KP_Code projects.

Overview Contact Subscription Settings

MVP highlights

  • Modular field system with hints, errors, required indicators, and inline a11y wiring.
  • Validation engine with required/email/length/custom hooks and top-level error summary.
  • State model: idle, loading, success, and error with retry pathway.
  • Progressive enhancement: semantic HTML works before JS; JavaScript upgrades behavior.

Quick start API

import { FormEngine } from './assets/js/modules/form-engine.js';

FormEngine.init(formElement, {
  validationRules: {
    email: { required: true, type: 'email' }
  },
  onSubmit: async (values) => ({ ok: true }),
  successMessage: 'Saved!',
  errorMessage: 'Please try again.'
});
