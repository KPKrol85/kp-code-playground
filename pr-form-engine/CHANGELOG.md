# Changelog

## 0.1.0 - Initial MVP
- Added Universal Form Engine project scaffold and documentation.
- Implemented modular form engine API: `FormEngine.init(formElement, options)`.
- Added validation engine with required, email, min/max length, and custom rule hooks.
- Added reusable UI helpers for inline errors, error summary, and form state transitions.
- Added accessibility module for aria-describedby wiring and first-error focus management.
- Added localStorage utility to persist last demo state.
- Added four demo pages: overview, contact, subscription, and settings.
- Added dark-first, token-based styles with reduced-motion support.
