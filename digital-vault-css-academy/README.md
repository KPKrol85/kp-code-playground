# KP_Code Digital Vault CSS Academy

## Czym jest ten projekt?
KP_Code Digital Vault CSS Academy to pierwszy, samodzielny produkt EDU w ekosystemie Digital Vault. Projekt został przygotowany jako nowoczesna, statyczna platforma do nauki CSS dla początkujących.

## Zakres obecnej wersji
Obecna wersja skupia się na statycznej, profesjonalnej stronie głównej platformy oraz pełnych czterdziestu dwóch lekcjach ścieżki CSS: od wprowadzenia do języka, przez selektory, box model, przepływ dokumentu, kolory i typografię, po Flexbox, CSS Grid, pozycjonowanie, warstwy z `z-index`, podstawy responsywności, media queries, jednostki CSS, obrazy responsywne, fluid typography oraz pierwsze komponenty UI z formularzami, kartami, listami, sekcjami treści, nawigacją, tabelami, danymi, separacją wizualną, transformacjami 2D, przejściami CSS, animacjami CSS, dostępnością ruchu, organizacją nazw klas, zmiennymi CSS, design tokens, kaskadą, specyficznością, `@layer`, resetem, normalize, stylami bazowymi, strukturą plików CSS, pseudoklasami, pseudoelementami, container queries, logical properties, CSS nesting, trybami kolorów, zmiennymi motywu, dark mode oraz pięcioma projektami końcowymi: responsywną kartą profilu, sekcją landing page, panelem dashboardu, formularzem rejestracji i mini systemem komponentów CSS.

Celem jest dostarczenie solidnego fundamentu pod dalszą rozbudowę o kolejne lekcje layoutu, moduły ćwiczeń i panele użytkownika.

## Technologie
- HTML5 (semantyczna struktura),
- CSS3 (mobile-first, design tokens, BEM-like naming),
- Vanilla JavaScript (progresywne ulepszenia UI).

Brak frameworków, build tooli i zewnętrznych zależności.

## Jak uruchomić lokalnie?
1. Otwórz folder `digital-vault-css-academy`.
2. Uruchom plik `index.html` w przeglądarce (dwuklik lub przez lokalny serwer statyczny).
3. Przejdź do wybranej lekcji z poziomu strony głównej.

## Kierunki dalszej rozbudowy
- kolejne lekcje CSS z naciskiem na responsywność, komponenty UI, animacje i organizację arkuszy,
- dashboard postępów nauki,
- moduł ćwiczeń i mini quizów,
- biblioteka gotowych snippetów CSS,
- ścieżka zaawansowana (architektura CSS i design systems).
