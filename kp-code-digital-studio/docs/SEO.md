# SEO Checklist

- Meta title + description na każdej stronie.
- OG + Twitter cards (użyj plików w /site/og).
- Canonical na każdej stronie.
- JSON-LD: Person, Organization/LocalBusiness, Service.
- Sitemap.xml + robots.txt.
- Favicon + manifest.

## OG Images
W folderze `/site/og` są placeholdery. Zastąp je grafikami 1200x630 px.

## Favicons
Zastąp `/site/assets/icons/favicon.svg` pełnym pakietem ikon, jeśli wymagane.
